# Garbarino App

This is a Front End project to test Javascript skills and App structuring. The project should display a given list of products and its information.


### Installation
```
$ git clone https://github.com/dieferrari/garbarinoApp

cd garbarinoApp

npm install
```


## Start & watch

```
npm start
```
